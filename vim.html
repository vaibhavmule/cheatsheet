vim cheatsheet

Note: Command starting from : is typed in Normal Mode.

  • h (left)
  • j (down)
  • k (up)
  • l (right)
  • <ESC> :q! <ENTER> to trash all changes.
  • <ESC> :wq <ENTER> to save the changes.
  • x To delete the character at the cursor type
  • i type inserted text <ESC>
  • A type appended text <ESC>
  • Pressing <ESC> will place you in Normal mode or will cancel an unwanted and partially completed command.
  • dw To delete from the cursor up to the next word
  • d$To delete from the cursor to the end of a line
  • ddTo delete a whole line
  • 0To move to the start of the line
  • uTo undo previous actions
  • UTo undo all the changes on a line
  • CTRL-RTo undo the undo's
  • pTo put back text that has just been deleted
  • rTo replace the character under the cursor
  • cTo change from the cursor to where the motion takes you. e.g. "c$", "ce"
  • CTRL-GTo display your location in the file and the file status
  • GTo move to end of the file
  • number GTo move to that line number
  • ggTo move to first line.
  • :/To search FORWARD for the phrase
  • :?To search BACKWARD for the phrase
  • nTo find the next occurrence in the same direction
  • NTo search in the opposite direction
  • CTRL-OTo take you back to older positions
  • CTRL-ITo take you back to newer positions
  • %To match (,),[,],{, or } and takes you it's match.
  • :s/old/newTo substitute new for the first old in a line
  • :s/old/new/gTo substitute new for all 'old's on a line
  • :#,#s/old/new/gTo substitute phrases between two line #'s
  • :%s/old/new/gTo substitute all occurrences in the file
  • :%s/old/new/gcTo ask for confirmation each time add 'c'

Most of the reference or command taken from "vimtutor". if you don't know what vimtutor is. Type "vimtutor" in your terminal.
